Requirements:-
    pavucontrol

Step 1:
    Run the script with the command:
    ```$ ./dynopii.sh```

Step 2:
    In the "Playback" tab of the pavucontrol window that opens, and change the playback device for the running "ALSA plug-in[aplay]: ALSA Playback on" and select 'dynopii' in the option present. 

Step 3:
    In your conferencing software, select 'Monitor of dynopii' as your audio input device.

And your microphone audio is now routed to the conference!
